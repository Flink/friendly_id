Overview

This class is not intended to be used on its own, it is used internally by FriendlyId#has_friendly_id to load the model’s configuration.

The arguments accepted by #has_friendly_id correspond to the writeable instance attributes of this class; please see the description of the attributes below for information on the possible options.

Examples:

has_friendly_id :name, 
 :use_slug => true,
 :max_length => 150, 
 :approximate_ascii => true,
 :ascii_approximation_options => :german,
 :sequence_separator => ":",
 :reserved_words => ["reserved", "words"],
 :scope => :country,
 :cache_column => :my_cache_column_name
 # etc.

def initialize(configured_class, method, options = nil, &block)
  @configured_class = configured_class
  @method = method.to_sym
  DEFAULTS.merge(options || {}).each do |key, value|
    self.send "#{key}=".to_sym, value
  end
  yield self if block_given?
end

- (Object) cache_column

The column used to cache the friendly_id string. If no column is specified, FriendlyId will look for a column named cached_slug and use it automatically if it exists. If for some reason you have a column named cached_slug but don’t want FriendlyId to modify it, pass the option :cache_column => false to #has_friendly_id.
